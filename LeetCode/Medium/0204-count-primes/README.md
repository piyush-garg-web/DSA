# Count Primes

Problem Link: https://leetcode.com/problems/count-primes

---

## Problem Statement

Given an integer n, return the number of prime numbers that are strictly less than n.

---

## Examples

### Example 1

```text
Input:
n = 10

Output:
4

Explanation:
There are 4 prime numbers less than 10, they are 2, 3, 5, 7.
Example 2:
```

### Example 2

```text
Input:
n = 0

Output:
0
```

### Example 3

```text
Input:
n = 1

Output:
0
```

---

## Constraints

```text
0 <= n <= 5 * 106
```
