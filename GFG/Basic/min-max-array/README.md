# Min Max Array

Problem Link: https://www.geeksforgeeks.org/problems/find-minimum-and-maximum-element-in-an-array4428/1

---

## Problem Statement

Given an array arr[]. Your task is to find the minimum and maximum elements in the array.

---

## Examples

### Example 1

```text
Input:
arr[] = [1, 4, 3, 5, 8, 6]

Output:
[1, 8]

Explanation:
minimum and maximum elements of array are 1 and 8.
```

### Example 2

```text
Input:
arr[] = [12, 3, 15, 7, 9]

Output:
[3, 15]

Explanation:
minimum and maximum element of array are 3 and 15.
```

---

## Constraints

```text
1 ≤ arr.size() ≤ 105
1 ≤ arr[i] ≤ 109
```
