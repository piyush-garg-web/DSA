# Middle Of Three

Problem Link: https://www.geeksforgeeks.org/problems/middle-of-three2926/1

---

## Problem Statement

Given three distinct numbers a, b and c. Find the number with a value in the middle (Try to do it with minimum comparisons).

---

## Examples

### Example 1

```text
Input:
a = 978, b = 518, c = 300

Output:
518

Explanation:
Since 518>300 and 518<978, so 518 is the middle element.
```

