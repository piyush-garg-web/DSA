# Value Equal Position

Problem Link: https://www.geeksforgeeks.org/problems/value-equal-to-index-value1330/1

---

## Problem Statement

Given an array arr[]. Find the elements whose value is equal to the position where they appear.

Note: There can be more than one element in the array which have the same value as its position. You need to include the position of every such element.

---

## Examples

### Example 1

```text
Input:
arr[] = [15, 2, 45, 4, 7]

Output:
[2, 4]

Explanation:
Here, arr[2] = 2 exists here and arr[4] = 4 exists.
```

### Example 2

```text
Input:
arr[] = [1]

Output:
[1]

Explanation:
Here arr[1] = 1 exists.
```

---

## Constraints

```text
1 ≤ arr.size ≤ 105
1 ≤ arr[i] ≤ 106
```
