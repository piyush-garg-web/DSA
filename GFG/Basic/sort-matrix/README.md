# Sort Matrix

---

## Problem Statement

Given an n*n matrix mat[][]. returns the matrix with all elements sorted in non-decreasing order.

---

## Examples

### Example 1

```text
Input:
mat[][] = [[10, 20, 30, 40],

Output:
[[10, 15, 20, 25],

Explanation:

Sorting the matrix gives this result.
```

### Example 2

```text
Input:
mat[][] = [[1, 5, 3],

Output:
[[1, 2, 3],

Explanation:

Sorting the matrix gives this result.
```

---

## Constraints

```text
1 ≤ n ≤ 1000
1 ≤ mat[i][j] ≤ 105
```
