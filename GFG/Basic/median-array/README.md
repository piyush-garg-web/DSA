# Median Array

Problem Link: https://www.geeksforgeeks.org/problems/find-the-median0527/1

---

## Problem Statement

Given an array arr[] of integers, calculate the median.

---

## Examples

### Example 1

```text
Input:
arr[] = [90, 100, 78, 89, 67]

Output:
89

Explanation:
After sorting the array middle element is the median
```

### Example 2

```text
Input:
arr[] = [56, 67, 30, 79]

Output:
61.5

Explanation:
In case of even number of elements, average of two middle elements is the median.
```

### Example 3

```text
Input:
arr[] = [1, 2]

Output:
1.5

Explanation:
The average of both elements will result in 1.5.
```

---

## Constraints

```text
1 <= arr.size() <= 105
1 <= arr[i] <= 105
```
