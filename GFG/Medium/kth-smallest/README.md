# Kth Smallest

Problem Link: https://www.geeksforgeeks.org/problems/kth-smallest-element5635/1

---

## Problem Statement

Given an integer array arr[] and an integer k, find and return the kth smallest element in the given array.
Note: The kth smallest element is determined based on the sorted order of the array.

---

## Examples

### Example 1

```text
Input:
arr[] = [10, 5, 4, 3, 48, 6, 2, 33, 53, 10], k = 4

Output:
5

Explanation:
4th smallest element in the given array is 5.
```

### Example 2

```text
Input:
arr[] = [7, 10, 4, 3, 20, 15], k = 3

Output:
7

Explanation:
3rd smallest element in the given array is 7.
```

---

## Constraints

```text
1 ≤ arr.size() ≤ 105
1 ≤ arr[i] ≤ 105
1 ≤ k ≤  arr.size()
```
