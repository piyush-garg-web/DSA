# Kth Smallest Element Matrix

Problem Link: https://www.geeksforgeeks.org/problems/kth-element-in-matrix/1

---

## Problem Statement

Given a matrix mat[][] of size n*n, where each row and column is sorted in non-decreasing order. Find the kth smallest element in the matrix.

---

## Examples

### Example 1

```text
Input:
mat[][] = [[16, 28, 60, 64], k = 3

Output:
27

Explanation:
27 is the 3rd smallest element.
```

### Example 2

```text
Input:
mat[][] = [[10, 20, 30, 40], k = 7

Output:
30

Explanation:
30 is the 7th smallest element.
```

---

## Constraints

```text
1 ≤ n ≤ 500
1 ≤ mat[i][j] ≤ 104
1 ≤ k ≤ n*n
```
