# Palindrome String

---

## Problem Statement

Given a string s, return true if the string is a palindrome. Otherwise, return false.

A string is considered a palindrome if it reads the same forwards and backwards.

---

## Examples

### Example 1

```text
Input:
s = "abba"

Output:
true

Explanation:
"abba" reads the same forwards and backwards, so it is a palindrome.
```

### Example 2

```text
Input:
s = "abc"

Output:
false

Explanation:
"abc" does not read the same forwards and backwards, so it is not a palindrome.
```

---

## Constraints

```text
1 ≤ s.size() ≤ 106
The string s contains only lowercase english letters (a-z).
```
